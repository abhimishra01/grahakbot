## src/utils/systemPrompt.js
Build a function buildSystemPrompt(config) that
takes a config object and returns a system prompt:

config shape:
{
  businessName: string,
  businessType: string,
  services: string[],
  hours: string,
  location: string,
  tone: 'friendly' | 'professional' | 'casual'
}

The prompt must:
- Set the AI as a customer assistant for that business
- List the services, hours, location
- Set tone based on selection
- Instruct replies to be 1-3 sentences max
- Sound like a real WhatsApp message
- Never mention being an AI unless directly asked
- Handle orders, appointments, FAQs, pricing queries
- Politely decline anything outside business scope

---

## src/utils/aiClient.js
Multi-provider client identical to project standard.
Single function: sendMessage(messages, config)
where messages is the full conversation history array
and config has providerKey and apiKey.

Each provider call passes:
- System prompt from buildSystemPrompt()
- Full conversation history for context
- Max tokens: 150 (keep replies short like WhatsApp)
- Try/catch with meaningful error messages
